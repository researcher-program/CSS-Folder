CSS Specificity And Priority

Specificity CSS

কোনো নিদিরষ্ট element/tag'র জন্যে একের অধিকবার যদি একই {Inline/Internal?External}CSS করি তাহলে কোন CSS টা কাজ করবে তা নির্ধারণ করে।

Priority CSS

এটি হল CSS এর element/tag'র জন্য কার অধিকার বেশি।

  1. 1st Priority -> Inline;
  2. 2nd Priority -> Internal;
  3. 3rd Priority -> External.

কিন্তু, এই 3টির যেকোনো 1টিকে বেশি Priority বুঝাতে "!Important" চিহ্নটি ব্যবহার করা হয়।
